#  Jogo de Pife 

##  Pife em C++ (Orientado a Objetos) 

 Este projeto implementa o popular jogo de cartas "Pife" (também conhecido como Pif Paf ou Cacheta) em C++, aplicando os princípios da Programação Orientada a Objetos (POO). O jogo simula a dinâmica de compra, descarte e formação de combinações (trincas e sequências) com o objetivo de "bater" primeiro, demonstrando o uso de classes e objetos para modelar as entidades e interações do jogo.

## Funcionalidades Principais:

    Partidas completas: Desde a distribuição das cartas até a vitória de um jogador.

    Interação de turno: Compra do monte/lixo e descarte.

    Validação de combinações: Suporte para trincas e sequências.
